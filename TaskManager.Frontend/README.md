This folder contains the Angular frontend of the TaskManager application.

## Prerequisites

- Node.js >= 20
- npm (or yarn/pnpm)
- Angular CLI 17+

## Setup

Install dependencies:

```bash
npm install
```

## Development Server
```bash
ng serve
```

Navigate to http://localhost:4200/. The app will automatically reload if you change any source files.

## Build
```bash
ng build --prod
```

The output will be in the dist/ folder.

## Testing
```bash
ng test
```

## Notes

Environment files are in src/environments/. Use environment.ts for development and environment.prod.ts for production.
