This folder contains the .NET backend of the TaskManager application.

## Prerequisites

- .NET 8 SDK
- Optional: IDE (Rider, Visual Studio)

## Setup

Restore dependencies:

```bash
dotnet restore
```

Build the project:
```bash
dotnet build
```

## Run
```bash
dotnet run
```
The API will be available at http://localhost:5000 by default.

## Testing
```bash
dotnet test
```

## Notes

Connection strings and secrets can be configured in appsettings.json or environment variables.
